*Windows key is interchangeable with the control key*
# Navigation

| **Function**                     | **Command**       |
| -------------------------------- | ----------------- |
| Change mode                      | `Ctrl + e`        |
| Quick Switcher                   | `Ctrl + o`        |
| Navigate tabs (forward)          | `Crtl + alt + <-` |
| Navigate tabs (backward)         | `Ctrl + alt + ->` |
| Open command palette             | `Ctrl + p`        |
| Open link under cursor (new tab) | `Ctrl + enter`    |

# Programming
| **Function**      | **Command**       |
| ----------------- | ----------------- |
| `cmd + shift + i` | Developer console |
# Custom Binds

| **Function**      | **Command**          |
| ----------------- | -------------------- |
| `Alt + f`         | Fold all             |
| `Alt + d`         | Unfold all           |
| `Alt + shift + f` | Fold more            |
| `Alt + shift + d` | Unfold more          |
| `crtl + s`        | strikethrough        |
| `crtl + h`        | highlight            |
| `crtl + m`        | inline math          |
| `crtl + u`        | insert markdown link |

# Aesthetic Formatting

| **Function** | **Command**  |
| ------------ | ------------ |
| `Win + .`    | insert emoji |
