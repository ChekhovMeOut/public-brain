# Stenography
- Depending on how my professional goals and tasks develop, it may become prudent to explore stenography
- The core of steno is to type phonetically rather than using characters. 
- The greatly reduce keyboard requires that multiple keys are pressed together to generate the full range of sounds. 
- The final key point is that steno is a customizable language- input from the keyboard is interpreted using a custom dictionary, allowing for unique modification of frequent input
- This [video](https://www.youtube.com/watch?v=7jVXrX5TDk4) provides a great introduction to the idea and sets realistic expectations
